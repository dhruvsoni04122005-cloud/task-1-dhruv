# To-Do List Program

## Description
This is a simple Python To-Do List application that allows users to:
- Add tasks
- View all tasks
- Exit the program

## Concepts Used
- Lists
- Loops
- Conditional Statements
- User Input

## How to Run
1. Open the file in Python.
2. Run the program.
3. Choose an option from the menu.
